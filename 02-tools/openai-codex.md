# OpenAI Codex

GPT-5.3-Codex launched in February 2026 as OpenAI's most capable agentic coding model. They claim it "was instrumental in creating itself," used to debug its own training. It's 25% faster than the previous version.

I've been testing it for the last month. It's competitive with [Claude Code](./claude-code.md) for most tasks. The plugin system with multi-agent v2 workflows is interesting for complex automation.

**Who it's for:** teams already invested in OpenAI's ecosystem, or those who want the ChatGPT integration.
