# The Others

**Windsurf:** Cheaper than [Cursor](./cursor.md), basically similar features. OpenAI bought them, so who knows what happens next. Worth trying if budget matters.

**Codeium:** The best free option. If you can't expense tools, start here.

**Amazon Q:** Only makes sense if you're all-in on AWS. The AWS-specific suggestions are actually good. February 2026 updates strengthened its integration with Lambda and CloudWatch.

**Amazon Kiro:** New IDE built around [spec-driven development](../05-workflows/spec-driven-development.md). Uses EARS notation for unambiguous requirements. Worth looking at if you're in the AWS ecosystem.

**Google Antigravity:** In public preview with free access to Claude Opus, Gemini models, and OpenAI models through a single interface. Google's classic strategy: enter late, price aggressively. Worth watching.

**Cline/Aider:** Open source, bring your own API keys. Great if you want control and don't mind more setup.
