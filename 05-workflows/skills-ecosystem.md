# Skills

Skills are packaged prompts + workflows that teach AI coding tools how to do specific tasks well, the most consequential change to AI coding tooling since agents themselves. They got their own folder because the ecosystem deserves more than one page.

**Go to → [folder 06 — Skills](../06-skills-and-memory/)** for the full treatment: what skills are, the libraries worth knowing (gstack, Superpowers, vendor skills, security skills), how to choose them for your project, how to write your own, and how to avoid getting burned by the security situation (which is bad).
