# Skills & Memory

Two related concerns folded into one folder because in practice they show up together. **Skills** package agent behavior — workflows, prompts, instructions — into reusable artifacts the model can discover and load. **Memory** is what the agent persists across sessions — decisions made, conventions learned, mistakes corrected. Skills tell the agent *how* to do something this session; memory tells it *what was true* last session. The reader who's serious about either ends up needing both.
